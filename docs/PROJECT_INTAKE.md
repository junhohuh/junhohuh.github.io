# Project Intake

## Project
Name: Personal Website
One-line description: I want to build a personal website, including my biographic information, resume, and personal projects.

## Why
What am I building?: I am building a personal website about myself.
Why do I want to build it?: I want to build it to be able to paste it into job applications and show recruiters.

## User
Who is this for?: For myself to keep and manage.
Just me, or other users too?: It will be used to show in job applications, so that people can click and access information about me.

## MVP
What are the 3–5 core things it must do?
1. It will have a clear landing page and introduces users to who I am.
2. It will have a short bio/about section about me, and my contact information.
3. It will include a projects section.
4. It will include a resume link.
5. It will have a clean visual polish.

## Non-goals
What is explicitly out of scope for now?
- Mobile native application.

Budget: N/A

Platforms: Can be accessed Web + Mobile
Any preferred tools/stack: Should be able to be accessed from web and mobile, and all core functions should be available.

What data, APIs, or external services might this need?
- Unsure

## Success
What would make this project feel successful?
- Must perform all core functionalities, and as a student, it should showcase my work and myself very clear, structured way.
